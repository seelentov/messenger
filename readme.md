# MESSENGER

Мобильный мессенджер с базовыми функциями:

- Просмотр аккаунта
- Изменение данных
- Отправка и принятия сообщений

Технологии:

- React
- RTK Query
- Redix Toolkit
- js-Cookie
- json-server
- Firebase
