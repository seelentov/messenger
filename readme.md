# MESSENGER

Мобильный мессенджер с базовыми функциями:

- Просмотр аккаунта
- Изменение данных
- Отправка и принятия сообщений

Технологии:

- React
- Redix Toolkit
- js-Cookie
- react-router-dom
- eslint
- Firebase Auth
- Cloud Firestore
- use-sound
